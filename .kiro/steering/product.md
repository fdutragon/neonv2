# Product Overview

Neon Multimarcas is a premium vehicle dealership catalog system with advanced search capabilities and integrated administration.

## Purpose
- Digital vehicle catalog for premium dealerships
- Enables customers to browse, filter, and express interest in vehicles
- Provides administrators with full inventory management

## Target Users
- **Visitors**: Browse vehicles, use advanced filters, submit contact forms
- **Admins**: Manage vehicle inventory, control featured vehicles, view metrics

## Key Features
- Vehicle catalog with advanced search and filtering
- Featured vehicles showcase on homepage
- Detailed vehicle pages with image galleries and specifications
- Contact form for customer interest
- Admin dashboard with metrics
- Full CRUD operations for vehicle management
- PWA support for offline functionality and mobile installation

## Language
- Primary language: Portuguese (Brazil)
- UI text, error messages, and documentation should be in Portuguese
